# 🍔 DevBurger

Aplicação web interativa para exibição e manipulação de um menu de hambúrgueres utilizando **JavaScript moderno**.

O projeto foi desenvolvido para praticar **manipulação de arrays, renderização dinâmica no DOM e construção de interfaces responsivas**.

---

## 📸 Preview

![Preview do projeto](assets/preview.gif)

---

## 🚀 Funcionalidades

* Exibir todos os produtos do menu
* Aplicar desconto em todos os itens
* Filtrar hambúrgueres veganos
* Calcular o valor total do menu
* Exibir estado vazio quando nenhum item corresponde ao filtro

---

## 🧠 Conceitos de JavaScript aplicados

* Manipulação de DOM
* Métodos de arrays:

  * `forEach`
  * `map`
  * `filter`
  * `reduce`
* Organização de funções
* Renderização dinâmica de componentes
* Tratamento de estado vazio na interface

---

## 🎨 Interface e experiência do usuário

* Layout em **grid responsivo**
* Adaptação automática para **desktop, tablet e mobile**
* Botões organizados em **grid adaptativo no mobile**
* Cards com **sombra suave e elevação no hover**
* Feedback visual de interação com `hover` e `active`
* Transições suaves para melhorar a experiência do usuário

## 🎯 Objetivo do projeto

Praticar conceitos fundamentais de **JavaScript intermediário**, incluindo manipulação de arrays, lógica de programação e atualização dinâmica da interface.

Além disso, o projeto também explora conceitos básicos de **responsividade e microinterações em CSS**.
